# 안심이 (Ansim) 프로젝트 웹사이트

시민이 중심이 되어 재난 안전 정보를 함께 만들어 나가는 **[안심이]** 프로젝트의 소개 웹페이지입니다.
데이터베이스 없이 HTML, CSS, JavaScript만으로 가볍고 빠르게 구축된 정적 웹사이트(Static Website)입니다.

## 🚀 주요 기능 및 구성
- **프로젝트 소개**: 안심이 프로젝트의 비전과 주요 산출물(재난가방 프로토타입 등) 안내
- **활동 발자취 (타임라인)**: 2012년 첫 시작부터 현재 밋앤핵, 풀씨 활동까지의 핵심 기록
- **밋앤핵(Meet&Hack) 상세 기록**: 재난 안전 데이터를 탐구하는 시민들의 모임과 인터뷰 상세 내역 (`meethack.html`)
- **반응형 디자인**: 모바일과 데스크톱 등 다양한 기기에 최적화된 모던 웹 디자인 (Glassmorphism 적용)

## 🛠 기술 스택
- **HTML5**: 시맨틱 웹 구조 적용
- **CSS3 (Vanilla)**: 별도 프레임워크 없는 커스텀 스타일링, 부드러운 애니메이션 효과
- **JavaScript (Vanilla)**: Intersection Observer를 활용한 스크롤 애니메이션 및 모바일 네비게이션 제어

## 💻 로컬에서 실행하는 방법
별도의 서버 구축이나 패키지 설치가 필요 없습니다.
1. 이 저장소를 로컬 컴퓨터에 클론(`git clone`)하거나 다운로드합니다.
2. `index.html` 파일을 더블 클릭하여 웹 브라우저에서 바로 실행합니다.
3. (권장) VS Code를 사용하신다면 `Live Server` 익스텐션을 통해 실시간으로 변경 사항을 확인하며 테스트할 수 있습니다.

## 🌐 배포 가이드
이 저장소는 정적 파일만으로 구성되어 있어 **GitHub Pages, Vercel, Netlify** 등의 무료 호스팅 서비스에 폴더를 올리기만 하면 즉시 배포할 수 있습니다.

### GitHub Pages로 배포하기
1. GitHub 저장소의 **Settings** 탭으로 이동합니다.
2. 왼쪽 메뉴에서 **Pages**를 클릭합니다.
3. **Build and deployment** 섹션의 Source를 `Deploy from a branch`로 설정합니다.
4. 브랜치를 `main` (또는 작업하신 기본 브랜치)으로 선택하고 저장(`Save`)합니다.
5. 몇 분 후 제공되는 URL을 통해 전 세계 어디서든 웹사이트에 접속할 수 있습니다.
